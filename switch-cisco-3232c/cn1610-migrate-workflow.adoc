= Workflow zur Migration von CN1610-Switches zu Nexus 3232C-Switches
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Befolgen Sie diese Workflow-Schritte, um Ihre CN1610-Switches auf Cisco Nexus 3232C-Switches zu migrieren.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Eins"]link:cn1610-migrate-to-3232c-overview.html["Migrationsanforderungen"]
[role="quick-margin-para"]
Prüfen Sie die Beispielinformationen zum Migrationsprozess.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Zwei"]link:cn1610-prepare-to-migrate.html["Bereiten Sie sich auf die Migration vor"]
[role="quick-margin-para"]
Bereiten Sie Ihre CN1610-Switches für die Migration auf Nexus 3232C-Switches vor.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Drei"]link:cn1610-replace-CL2.html["Tauschen Sie den Schalter CL2 im Kombiinstrument aus."]
[role="quick-margin-para"]
Ersetzen Sie den Cluster-Schalter CL2 durch den neuen Nexus 3232C-Schalter C2.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["Vier"]link:cn1610-replace-CL1.html["Tauschen Sie den Schalter CL1 in der Instrumententafel aus."]
[role="quick-margin-para"]
Ersetzen Sie den Cluster-Schalter CL1 durch den neuen Nexus 3232C-Schalter C1.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["Fünf"]link:cn1610-complete-migration.html["Schließen Sie Ihre Migration ab."]
[role="quick-margin-para"]
Schließen Sie Ihre Migration auf die neuen Nexus 3232C Switches ab.
