= Überblick über den Gesundheitscheck
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Gesundheitsmonitore überwachen proaktiv bestimmte kritische Zustände in Ihrem Cluster und geben Warnmeldungen aus, wenn sie einen Fehler oder ein Risiko erkennen.

Um die aktuell ausgelösten Warnmeldungen des Ethernet-Switch-Integritätsmonitors anzuzeigen, führen Sie folgenden Befehl aus: `system health alert show -monitor ethernet-switch`

Um die verfügbaren Warnmeldungen des Ethernet-Switch-Integritätsmonitors anzuzeigen, führen Sie folgenden Befehl aus: `system health alert definition show -monitor ethernet-switch`
