= Workflow zur Migration von schalterlosen Clustern zu Zwei-Knoten-Clustern mit Schaltern
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Befolgen Sie diese Workflow-Schritte, um von einem Zwei-Knoten-Cluster ohne Switches zu einem Zwei-Knoten-Cluster mit Switches zu migrieren, der Cisco Nexus 3132Q-V Cluster-Netzwerk-Switches enthält.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["Eins"]link:migrate-switchless-to-switched-requirements.html["Migrationsanforderungen"]
[role="quick-margin-para"]
Prüfen Sie die Anforderungen und Beispielinformationen zum Migrationsprozess.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Zwei"]link:migrate-switchless-prepare-to-migrate.html["Bereiten Sie sich auf die Migration vor"]
[role="quick-margin-para"]
Bereiten Sie Ihre switchlosen Cluster auf die Migration zu Zwei-Knoten-Switch-Clustern vor.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Drei"]link:migrate-switchless-configure-ports.html["Konfigurieren Sie Ihre Ports"]
[role="quick-margin-para"]
Konfigurieren Sie Ihre Ports für die Migration von Zwei-Knoten-Clustern ohne Switches zu Zwei-Knoten-Clustern mit Switches.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["Vier"]link:migrate-switchless-complete-migration.html["Schließen Sie Ihre Migration ab."]
[role="quick-margin-para"]
Schließen Sie Ihre Migration von Clustern ohne Switches zu Clustern mit zwei Knoten und Switches ab.
