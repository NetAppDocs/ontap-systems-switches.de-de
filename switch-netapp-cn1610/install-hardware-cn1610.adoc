= Installieren Sie die Hardware für den NetApp CN1610-Switch
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Um die NetApp CN1610 Switch-Hardware zu installieren, folgen Sie den Anweisungen in einem der folgenden Handbücher.

* https://library.netapp.com/ecm/ecm_download_file/ECMP1117853["1G-Installationsanleitung"^].
+
Ein Überblick über die Hardware- und Softwarefunktionen des CN1601-Switches sowie den Installationsprozess.

* https://library.netapp.com/ecm/ecm_download_file/ECMP1117824["10G Installationsanleitung"^]
+
Ein Überblick über die Hardware- und Softwarefunktionen des CN1610-Switches sowie eine Beschreibung der Funktionen zur Installation des Switches und zum Zugriff auf die CLI.
